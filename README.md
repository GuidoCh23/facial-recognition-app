# Facial Recognition App

Aplicación Android de reconocimiento facial desarrollada en Kotlin.

## Requisitos

- Android Studio (versión reciente)
- Android SDK (se instala automáticamente con Android Studio)

## Cómo ejecutar el proyecto

1. Clona el repositorio:
   ```bash
   git clone https://github.com/GuidoCh23/facial-recognition-app.git
   ```

2. Abre Android Studio y selecciona **"Open"**, luego elige la carpeta del proyecto.

3. Espera a que Gradle sincronice las dependencias (ocurre automáticamente).

4. Conecta un dispositivo físico o inicia un emulador y presiona **Run**.

> `local.properties` no está incluido en el repositorio, pero Android Studio lo genera automáticamente al abrir el proyecto.
